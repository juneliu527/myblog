# 缩剪要求
1. 开头要有苏州自然+人文
2. 公共阅读区，儿童阅读区，设计馆，5G 体验馆，音像馆
3. 最大宣传点：人工智能，一键下单
4. 结尾
5. 人文转室内，要有转折点
